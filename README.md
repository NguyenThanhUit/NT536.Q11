Cách cấu hình: Làm theo hướng dẫn trong file

Apply cấu hình nginx.conf theo đường dẫn: sudo nano /usr/local/nginx/conf/nginx.conf

Tạo các file tại đường dẫn: /var/www/html/vod.html

Tạo thư mục: /var/www/html/livestream xong tạo file index.html

Thay đổi nội dung file bằng nội dung trên

Cách truy cập video livestream: Stream trên obs, key nhập tùy thích, vào đường dẫn: <ip máy ảo>/livestream/ là được
